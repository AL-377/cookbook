## Steps to create an assistant

1. modify the content in `create_assistant.py`
2. run `python create_assistant.py` to get the id generated by openai which will be stored in  `assistants.json`
3. modify the openai key and assistant id in `.env` file so as to specify the assistant to use
4. `chainlit run app.py -w`

**Reference:** [chainlit-cookbook](https://github.com/AL-377/cookbook)
